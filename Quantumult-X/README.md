# TikTok

#### 前言介绍
通过配置下面文中的规则配置,实现免拔卡,免翻墙使用tiktok,转载本文请标明出处,谢谢合作


#### 版本功能
支持最新版tiktok 18.3.0，可看直播，优化推送的直播与地区


### 使用方法

1.  点击配置文件--点击Raw--跳转后拷贝浏览器输入框的链接
2.  打开QuantumultX--找到重写--点击引用--右上角添加
3.  粘贴链接-右上角确定
